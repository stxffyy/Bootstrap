Twitter Bootstrap
Свободный набор инстурментов для создания сайто и веб-приложений
Включает в себя HTML и CSS шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейсов, включая JS. Сетки - заранее заданные размеры кнопок, которые можно сразу же использовать, например, ширина колонки 140px относится к класссу span2, который можно использовать в CSS описании документа. Типографика - описания шрифтов, определение некоторых классов для шрифтов, таких как код, цитаты и тп. Медиа - представляет некоторое управление изображениями и видео.
